직군 별 자격요건&우대사항 정보를 크롤링하여 빈출 키워드를 선정하고 이를 서비스


1. 원티드에서 직군별 분류 & 분류 코드 정리 
2. 각 직군 별 분류되어 있는 채용 공고들의 URL 취합 정리 
3. 각 채용 공고 URL에 접속하여 기업명/직군분류/직군 상세/주요업무/자격요건/우대사항 정보 크롤링
4. 크롤링 된 정보(줄 글 단위의 정제되지 않은 데이터)는 MongoDB에 저장 

5. MongoDB에 저장된 데이터를 파이썬으로 호출하여 주요업무, 자격요건, 우대사항에 관한 자연어처리
   - 주요업무, 자격요건, 우대사항에 자주 언급되는 경험, 역량 등을 파악
   - 위 데이터를 명사 / 형용사로 형태로 전처리를 진행
   + (신입, 경력 빈도 수 체크)
   + (영어-> 대소문자가 다르게 인식되어 분리된 빈도수 합산하기)
   + ('/' 구분자로 묶여 한 단어로 인식된 단어 분리하기)
   + (한글 형용사 형태 추가 / 지금은 명사만 있음)
   + 불용어 추가해주는 함수 & py 파일 만들기 
   + keyword DB에 모든 단어 저장하지 않고, [직군/단어/빈도수]를 누적으로 저장할 수 있는 방법 생각해볼 것!
